# ⚡ Sector 1: Energy

This directory contains scripts and data for emissions resulting from fuel combustion and fugitive emissions from fuels.

## Structure

- `1a_fuel_consumption/`: Emissions from the combustion of fuels (e.g., electricity production, transport).
- `1b_fugitive/`: Intentional or unintentional release of greenhouse gases during the production, transformation, and distribution of fossil fuels.

## Methodology

Calculations in this sector heavily rely on the **Gigajoule (GJ)** normalization. Ensure all activity data is converted to GJ before running scripts in this folder.
